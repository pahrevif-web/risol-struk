iPhone: ini terbuka di “Preview File”, bukan Safari. Di mode preview, JavaScript & Print sering tidak berjalan normal, jadi struk bisa kosong/aneh.

Solusi:
  1. Tap ikon Share (kotak panah ke atas)
  2. Pilih Open in Safari / Buka di Safari
Jika tidak ada “Open in Safari”: di Share Sheet pilih Copy, lalu buka Safari dan paste. Paling stabil: upload file jadi link (GitHub Pages/Netlify).

Struk • Print → Save as PDF

iPhone: pastikan buka di Safari (bukan preview file). Jika ada opsi, aktifkan Scale: 100% / nonaktifkan “Fit to page”.
